关于本站点

本站点是对 github 作为爬取数据展示站点的一个尝试。包括 B 站视频列表和播放数的历史数据。

简单介绍一下本站工作原理：

  • 使用git pages托管静态站点代码。所有站点页面代码在nettlemeow.github.io代码库的 master 分支下
  • 使用 travis 自动爬取 B 站 API 数据(spider 目录下),并将结果git push到nettlemeow-data代码库下。
  • 使用 travis 将最新数据推送到 git pages 所属分支。

本站代码修改 up 主 id 后可用于 B 站任一 up 主的历史视频播放数的记录。
